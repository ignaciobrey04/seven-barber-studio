# Fotos de cortes

Colocá acá las fotos reales de la barbería con estos nombres exactos
(la web las toma automáticamente; mientras no estén, se muestra un placeholder elegante):

- `corte-1.jpg`  → corte fade / nuca (foto de perfil del cliente)
- `corte-2.jpg`  → corte con flequillo + fade
- `corte-3.jpg`  → interior / fachada del local (los hexágonos de luz)
- `corte-4.jpg`  → detalle de perfilado / otro corte

**Recomendado:** formato vertical (4:5), mínimo 800×1000px, JPG optimizado (<300 KB c/u).
Podés usar TinyPNG o Squoosh para comprimir sin perder calidad.
